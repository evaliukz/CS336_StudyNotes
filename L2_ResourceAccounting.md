# 1️⃣ 什么是 Memory accounting？

一句话：

算清楚“内存都被谁吃掉了”

在深度学习里，内存主要被下面几类东西占用：

模型参数（weights）

中间激活（activations）

梯度（gradients）

优化器状态（optimizer states）

临时 buffer（算子内部用）

很多人只算参数，这是最常见误区。

### 大白话例子

假设你有一个模型：

参数：1B（10 亿参数）

dtype：float16（2 bytes）

❌ 错误理解

模型大小 = 1B × 2B = 2GB
我的 GPU 16GB，稳了！

✅ 真实情况（训练时）：👉 直接爆显存

| 项目               | 内存     |
| ---------------- | ------ |
| 参数               | 2GB    |
| 梯度               | 2GB    |
| Adam 优化器（m + v）  | 4GB    |
| 激活（可能是参数的 2～5 倍） | 4–10GB |
